css弹性盒

css弹性盒

以下属性写在父元素里

怪异盒模型

box-sizing:border-box;

写边框和padding值 宽高不变

宽高保持不变 边框和padding加到了盒子里面 最终content区域变小

box-sizing:content-box;

转化为默认效果 普通盒模型

弹性盒 以下属性用在父元素中

弹性盒内的子元素是行内元素的话会转换成块元素

对子元素的子元素不生效

display:flex;

子元素默认情况下横向显示

display：inline-flex；

行内弹性盒

主轴对齐方式

justify-content:flex-start; 开头

justify-content:flex-end; 尾部

justify-content:center; 中间

justify-content:space-between; 空间分布在元素中间

justify-content:space-around; 空间分布在元素两边

justify-content:space-evenly; 空间平均分布

侧轴对齐方式

align-items:flex-start

align-items:flex-end

align-items:center

baseline

行内元素文字基线对齐

flex-direction:; 主轴方向

row 横向 默认值 reverse

column 竖向

flex-wrap：； 换行 reverse

no-wrap； 默认 不换行

wrap 换行

align-content：； 行与行之间的对齐方式 多轴对齐方式

flex-start 开头

flex-end 尾部

center 中间

space-between 空间分布在元素中间

space-around 空间分布在元素两边

以下属性写在子元素里

align-self ：； 某元素的对齐方式（侧轴）

               flex-start

               flex-end

               center

order：； 子元素排列顺序

0 默认值

1 往后

-1 往前

flex-grow ：； 生长 分配剩余空间 默认为0

flex-shrink ：； 压缩 收缩 元素空间收缩 默认为1

flex-basis：；为元素分配空间 默认为auto

flex：none； 默认值

flex：1； 相当于 flex-grow：1； 用于平均分配剩余空间（导航元素空间一致）

flex：auto； 适用于nav中文字不一致，文字间距一样 此时不写a的宽度
